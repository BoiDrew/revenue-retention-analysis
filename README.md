# Title: Revenue & Retention Analysis
## Description: 
Analyzing customer revenue patterns, retention, and cohorts using SQL and Python.

## Business Questions
1. How does revenue change over time?
2. What percentage of customers return after their first purchase?
3. Which customer cohorts generate the most long-term revenue?
4. When do customers typically churn?


## Dataset
Online Retail transactional dataset from UCI Machine Learning Repository.
Each row represents a product-level transaction, used to analyze revenue trends and customer retention behavior.
